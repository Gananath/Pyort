# Pyort
Command line tool for monetoring all network connections

# Installation
`pip install git+https://github.com/Gananath/Pyort.git`

# Usage
Help: `pyort -h`
Start: `pyort --start`
Custom: `pyort -s --kind tcp6`

# Configuration
`Config.ini` and database files exsists in ure `$HOME/.config/pyort`
